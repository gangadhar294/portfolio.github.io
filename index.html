// SYSTEM OVERVIEW

This project implements the core concepts learned in the Design and Analysis of Algorithms (DAA) course. We explored how to solve complex real-world problems using mathematical modeling and efficiency analysis.

Module A

Algorithmic Efficiency

Optimizing computational resources (Time & Memory) for large scale urban datasets. Includes Big-O analysis and performance tuning.

Module B

Graph Theory

Modeling city infrastructure using Nodes and Edges to solve Shortest Path (Dijkstra) and Connectivity (MST) problems.

Module C

Optimization

Using Dynamic Programming and Greedy approaches to solve resource allocation and scheduling conflicts in the city grid.

// BLUEPRINT MAP

Our team identified critical urban infrastructure challenges—Traffic, Water, and Power—and mapped them to standard Data Structures.
