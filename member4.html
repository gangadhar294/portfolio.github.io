Dijkstra's Algo

GreenCorridor Protocol

Prioritizing emergency vehicles using dynamic path weights.

▼

The Concept

Ambulances often get stuck at the "KL Rahul Stadium" junction. Normal traffic lights are static. We need a system that detects an ambulance and turns lights Green along the shortest path.

The Logic

INPUT: Intersection Graph, Traffic Density
WHILE Ambulance is active:
  1. Get CurrentLocation & HospitalDest
  2. Apply Dijkstra to find path with min (Density * Distance)
  3. Override Signals to GREEN for nodes in Path[]
RETURN Safe Route

HydroGrid Network

Zero-leakage water distribution for remote zones.

▼

The Concept

Connecting the "High Altitude" zone to the "Water Treatment Plant" is costly. We need to connect every residential node using the minimum total length of pipes possible.

The Logic

INPUT: Zones (Vertices), PipeCost (Edges)
FUNCTION BuildNetwork():
  Start at SourceNode
  REPEAT until all zones connected:
    1. Find cheapest pipe connecting ConnectedZone to NewZone
    2. Add pipe to Network
    3. Ensure NoCycles (Loops)

RapidExit System

Tsunami evacuation routing from Port areas.

▼

The Concept

When a Tsunami warning hits the Port (Low-lying), we need to guide crowds to the High Altitude farm. BFS helps us create a "Layered" escape plan, moving people 1 street away, then 2 streets away, avoiding jams.

The Logic

INPUT: CityMap, DangerSource
FUNCTION Evacuate():
  Queue Q.add(Port)
  WHILE Q not empty:
    curr = Q.remove()
    FOR each neighbor:
      Calculate HopsFromDanger
      Assign Direction Arrow -> Away

CargoMax Optimizer

Maximizing profit with limited port storage.

▼

The Concept

The Port has limited storage space (Capacity). Hundreds of containers arrive with different Profits and Weights. We must select the best subset to maximize revenue.

The Logic

INPUT: Containers[], Capacity W
FUNCTION SolveDP(item, remaining_cap):
  IF weight > remaining_cap: Skip Item
  ELSE:
    Option1 = Profit + SolveDP(next, cap - weight)
    Option2 = SolveDP(next, cap)
    RETURN MAX(Option1, Option2)

SwiftLink Delivery

Last-mile optimization for multiple stops.

▼

The Concept

A delivery truck starts at the Industrial Warehouse and must drop parcels at the University, Market, and Residential Zones in one trip, minimizing fuel consumption.

The Logic

INPUT: DeliveryLocations[]
GOAL: Find Hamiltonian Cycle with Min Cost
LOGIC:
  Check all permutations of stops:
  Warehouse -> A -> B -> C -> Warehouse
  Warehouse -> B -> A -> C -> Warehouse
  Select MinDistance path.
