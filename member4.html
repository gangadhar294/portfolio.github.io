1. GreenCorridor Protocol

Dijkstra's Algorithm

Problem: Emergency vehicles face unpredictable delays near the stadium during events.

Strategy: We model intersections as nodes and congestion as edge weights. The system calculates the path with minimum weight (time) and overrides traffic signals.

Input: Graph G(V, E), Source S
function FindPath(S, Target):
  PQ = PriorityQueue()
  PQ.push(S, 0)
  while PQ not empty:
    u = PQ.extractMin()
    for neighbor v of u:
      if dist[u] + weight < dist[v]:
        dist[v] = dist[u] + weight
        path[v] = u
return path

2. HydroGrid Network

Prim's MST

Problem: High infrastructure cost to connect remote zones (Farming/High Altitude) to water sources.

Strategy: We treat zones as vertices. We must connect all vertices using edges (pipes) such that the total cost is minimized and no cycles are formed.

Input: Zones [], Cost Matrix[][]
function ConnectWater():
  MST_Set = {StartNode}
  while MST_Set != AllNodes:
    Select edge (u, v) with min weight
    where u in MST_Set and v not in MST_Set
    Add v to MST_Set
    TotalCost += weight(u, v)

3. RapidExit Evacuation

BFS (Breadth-First Search)

Problem: Safe and orderly evacuation from the Port (Low-lying) to High Ground during Tsunami warnings.

Strategy: BFS explores the map layer-by-layer. It guarantees finding the route with the fewest "hops" (turns/streets) to safety.

Input: CityGrid, DangerSource (Port)
function PlanRoute():
  Queue Q.push(Port)
  Mark Port as Visited
  while Q not empty:
    current = Q.pop()
    for neighbor n in AdjacencyList:
      if n is SafeZone: Found Exit
      Q.push(n)

4. CargoMax Optimizer

0/1 Knapsack (DP)

Problem: Limited storage capacity at the Port. We must choose which containers to store to maximize profit.

Strategy: Used Dynamic Programming to test every combination of containers efficiently without recalculating overlapping sub-problems.

Input: Weights[], Profits[], Capacity W
function MaxProfit(n, W):
  if w[n] > W:
    return MaxProfit(n-1, W)
  else:
    return max(
      Profit[n] + MaxProfit(n-1, W-w[n]),
      MaxProfit(n-1, W)
    )

5. SwiftLink Delivery

TSP (Backtracking)

Problem: E-commerce vans need to visit Market, University, and Homes in one trip with minimum fuel.

Strategy: Modeled as the Traveling Salesman Problem. We find a Hamiltonian cycle with minimum edge weight sum.

Input: Graph of Delivery Points
function MinRoute(curr, visited):
  if all visited:
    return cost(curr -> start)
  min_cost = Infinity
  for city in neighbors:
    if not visited:
      cost = dist + MinRoute(city)
      min_cost = min(min_cost, cost)
return min_cost
