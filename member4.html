GreenCorridor

TIME: O(E log V)
The Problem

Traffic congestion at the "Stadium Junction" causes 400% delays. Ambulances from the Industrial Zone were getting stuck, risking patient survival.

The Solution

We implemented Dijkstra’s Algorithm. Instead of just distance, we calculate "Edge Weight" using real-time traffic density sensors. This guarantees the mathematically fastest route, not just the shortest.

Pseudocode Logic
function Dijkstra(Graph, source): dist[source] = 0 create priority_queue Q Q.add(source) while Q is not empty: u = Q.extract_min() for each neighbor v of u: alt = dist[u] + length(u, v) if alt < dist[v]: dist[v] = alt prev[v] = u Q.add(v)

HydroGrid

SPACE: O(V)
The Problem

Laying water pipes to the "High Altitude Zone" is expensive (₹10k/meter). We needed to connect all residential zones to the main tank with the minimum total pipe length.

The Solution

We utilized Prim’s Algorithm to build a Minimum Spanning Tree (MST). This ensures 100% connectivity with zero redundant loops, saving ₹2.4 Crores in estimated construction costs.

Pseudocode Logic
function Prims(Graph): MST_Set = {false...} Keys = {infinite...} Keys[0] = 0 for count 0 to V-1: u = minKey(Keys, MST_Set) MST_Set[u] = true for v in adj[u]: if graph[u][v] < Keys[v]: Keys[v] = graph[u][v] parent[v] = u

RapidExit

TIME: O(V + E)
The Problem

The Port and Beach zones are Tsunami-prone. During a panic, standard GPS sends everyone to the main highway, causing fatal gridlock.

The Solution

Modeled as an unweighted grid, we use Breadth-First Search (BFS). It explores the city layer-by-layer, finding the nearest exit for thousands of citizens simultaneously.

Pseudocode Logic
function BFS(start_node): create queue Q Q.enqueue(start_node) visited[start_node] = true while Q is not empty: t = Q.dequeue() if t is SafeZone: return path for neighbor in t: if not visited[neighbor]: visited[neighbor] = true Q.enqueue(neighbor)

CargoMax

DP TABLE
The Problem

Ships arrive with mixed cargo (High Value/Low Weight vs Low Value/High Weight). The port has a fixed weight capacity. We must maximize profit.

The Solution

Greedy fails here. We use 0/1 Knapsack (Dynamic Programming). We build a 2D table to calculate the max value for every possible weight limit up to capacity.

Pseudocode Logic
for i from 0 to n: for w from 0 to W: if weight[i-1] <= w: dp[i][w] = max( val[i-1] + dp[i-1][w-weight[i-1]], dp[i-1][w] ) else: dp[i][w] = dp[i-1][w]
