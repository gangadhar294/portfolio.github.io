GreenCorridor Protocol

Dijkstra
The Challenge

Emergency vehicles face unpredictable delays at the Stadium Junction. Static signals are inefficient for critical response times.

Algorithmic Logic
> INPUT: Graph(Intersections), Weights(Traffic) > INIT: Priority Queue PQ, Distances=∞ > WHILE PQ not empty: u = PQ.extractMin() FOR v in neighbors(u): IF dist[u] + weight(u,v) < dist[v]: dist[v] = new_dist Update Path > OUTPUT: Dynamic Green Wave path

HydroGrid Network

Prim's MST
The Challenge

Connecting remote high-altitude zones to water sources is cost-prohibitive. We must minimize total pipe length without creating redundant loops.

Algorithmic Logic
> INPUT: Vertices(Zones), Edges(Pipe Costs) > INIT: MST_Set = {Start_Node}, MinHeap > WHILE MST_Set != All_Vertices: Extract min cost edge (u, v) WHERE u is in MST, v is NOT Add v to MST_Set TotalCost += EdgeWeight > OUTPUT: Minimum Cost Piping Layout

RapidExit System

BFS
The Challenge

The Port area is a Tsunami risk zone. Unorganized evacuation leads to bottlenecks. We need a layered, "fewest-hops" escape plan.

Algorithmic Logic
> INPUT: City Grid, Source(Port) > INIT: Queue Q.push(Source), Visited[] > WHILE Q not empty: curr = Q.dequeue() FOR neighbor n of curr: IF n not Visited: Mark Visited Set Direction -> Away from Source Q.enqueue(n) > OUTPUT: Layered Evacuation Map

CargoMax Optimizer

Knapsack DP
The Challenge

Limited port storage capacity vs. incoming containers of varied profit and size. We must select the optimal subset for maximum revenue.

Algorithmic Logic
> INPUT: Profits[], Weights[], Capacity C > CREATE DP Table[Items+1][C+1] > FOR i = 1 to Items: FOR w = 1 to C: IF Weights[i] <= w: DP[i][w] = max(Profits[i] + DP[i-1][w-Wi], DP[i-1][w]) ELSE: DP[i][w] = DP[i-1][w] > OUTPUT: Max Profit Value

SwiftLink Delivery

TSP Graph
The Challenge

Last-mile delivery requires visiting multiple distinct zones (Market, University, Home) in a single trip with minimum fuel consumption.

Algorithmic Logic
> INPUT: Graph of Delivery Nodes > GOAL: Find min cost Hamiltonian Cycle > APPROACH (Dynamic Programming): Mask = Set of visited cities DP(mask, pos) = min cost to visit set 'mask' ending at 'pos' Iterate through all states to find min total path. > OUTPUT: Optimal Stop Order
