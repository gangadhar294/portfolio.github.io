GreenCorridor Protocol

Dijkstra
The Business Case

Traffic congestion between the Industrial Area and the Hospital causes critical delays, especially during Stadium events where travel time spikes by 400%. We needed a fail-safe navigation system to prioritize emergency vehicles.

Graph Modeling

Nodes: Road Intersections.
Edges: Road Segments.
Weights: Dynamic Cost (Distance × Traffic Density).

Algorithm Analysis

Dijkstra’s Algorithm was chosen over BFS because our graph contains weighted edges (traffic delays). It guarantees the mathematical shortest path in O(E log V) time.

Implementation Logic
> INIT all distances = Infinity, Source = 0 > PUSH Source to Min-Priority Queue > WHILE Queue not empty: u = Extract Min FOR v in neighbors(u): IF Time[u] + RoadWeight < Time[v]: Time[v] = New_Time Parent[v] = u > BACKTRACK from Hospital to Source.

HydroGrid Network

Prim's MST
The Business Case

Connecting the remote High Altitude Zone to the water supply is capital intensive (₹10k/meter). We optimized the pipeline network to ensure 100% coverage with the minimum possible infrastructure cost.

Graph Modeling

Nodes: City Zones (Residential, Zoo, etc).
Edges: Potential Pipeline Routes.
Weights: Excavation Cost.

Algorithm Analysis

Prim’s Algorithm builds a Minimum Spanning Tree (MST). This ensures every zone is connected without redundant loops, maintaining water pressure and minimizing cost.

Implementation Logic
> START at Water Treatment Plant > MAINTAIN Min-Heap of pipe connections > WHILE Visited_Count < Total_Zones: Pick pipe with MIN cost connecting Visited -> Unvisited Add pipe to Network Mark new zone as Visited > RESULT: Optimized Infrastructure.

RapidExit System

BFS
The Business Case

The Port and Beach zones are high-risk tsunami areas. Standard GPS routing creates bottlenecks. We implemented a multi-directional evacuation model to guide crowds to the nearest safe zone (High Altitude Area).

Graph Modeling

Grid Graph: The city is mapped as a coordinate grid. Goal: Find the path with the fewest "hops" (intersections) to safety.

Algorithm Analysis

Breadth-First Search (BFS) explores layer-by-layer, mimicking the physical spread of a flood. It guarantees the shortest path in an unweighted grid. Complexity: O(V + E).

Implementation Logic
> ENQUEUE Danger Zone (Port) > MARK as Visited > WHILE Queue not empty: Current = Dequeue CHECK neighbors (N, S, E, W) IF Unvisited: Record Distance from Danger Set Flow Direction -> Away Enqueue Neighbor > GENERATE Heat Map.

CargoMax Optimizer

Knapsack DP
The Business Case

The Port storage yard has limited weight capacity. We handle mixed cargo: High-value Electronics (Light) vs Low-value Raw Materials (Heavy). We must select the optimal mix to maximize revenue without overloading.

Graph Modeling

Combinatorial Optimization:
Items = Containers.
Constraint = Max Capacity.
Objective = Max Profit.

Algorithm Analysis

0/1 Knapsack (Dynamic Programming). A greedy approach fails here. DP explores all "Take it or Leave it" scenarios to find the mathematical optimum.

Implementation Logic
> CREATE Table DP[Item][Capacity] > FOR each Container: IF Weight > Current_Capacity: Value = Previous_State ELSE: Value = MAX( Profit + DP[Prev_Item][Rem_Cap], DP[Prev_Item][Current_Cap] ) > CELL[N][W] contains Max Profit.

SwiftLink Delivery

TSP Graph
The Business Case

Logistics trucks must visit the University, Market, and Residential zones from the Warehouse in a single trip. Random routing wastes fuel. We calculated the single most efficient loop.

Graph Modeling

Complete Weighted Graph: Every location connects to every other location. Goal: Hamiltonian Cycle with minimum weight.

Algorithm Analysis

Traveling Salesman Problem (TSP). For our set of N locations, we used the Held-Karp algorithm (DP + Bitmasking) for exact precision. Complexity: O(N^2 * 2^N).

Implementation Logic
> REPRESENT Visited Set as Bitmask > DP(mask, pos) = Min cost to visit set 'mask' ending at 'pos' > TRANSITION: Try moving to every unvisited city 'next' Recursive Call: DP(mask | next, next) > ADD return distance to Warehouse.
